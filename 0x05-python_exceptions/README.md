# 0x05-python_exceptions
Exception in python are used to hadle errors in pyython programming.
some examples of errors in python programming are as follows:
- TypeError
- NameError
- IndexError
- IndentaionError
among many other

## Syntax
try:
	block of code
except {exception}:
	block of code
```
try:
	my_list = [1, 2, 3, 4, 5]
	print(my_list[8])
except IndexError:
	print("Index out of range")
```
